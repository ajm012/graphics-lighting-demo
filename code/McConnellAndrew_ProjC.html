CONTROLS:

Use the WSAD keys to turn the camera, the arrow keys to move forward/backward/left/right, and the OP keys to move up and down.

Use the M key to cycle through materials, and the keys 1-6 to toggle the ambient/diffuse/specular values of the two light sources on/off.

Use the IKJL keys similar to WSAD: I moves the point-source light up (+y), K moves it down, J moves it left (-x), L moves it right. Y moves it in the -z direction, U in the +z.

Enjoy!
